Horus v1.8 — Manual Tests

Ready for testing · Reference: v1.7.2 (stable)

WHAT'S NEW IN v1.8 — Changes from v1.7.2

  • Admin → Tour Catalog: new section to manage the 14 tours (initialization, CRUD price, active/inactive toggle, delete, manual add)
  • Admin → New Route: add tour form with free-text category (free input + suggestions)
  • Booking form (dashboard + calendar): tour select filtered by platform — auto-fills name, duration, route type, price
  • Civitatis — per-person pricing: formula 60€ × N pers. = total shown live in the form
  • BookingCard: price shown in green on each card · per-person breakdown for Civitatis
  • Calendar — side panel: "Precio" row visible when a booking has a price
  • Stats → Most booked tours: platform badges (Viator / GYG / Civitatis…) shown under each tour
  • Demo seed data: requires catalog to be initialized first · uses real names and IDs from the catalog
